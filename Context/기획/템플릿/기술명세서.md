**예시**
```json

  "field1": "value1",
  "field2": 123,
```

**응답 (Response)**

**성공 (200 OK)**
```typescript
interface SuccessResponse {
  success: true;
  data: {
    result: string;
    timestamp: string;
  };
}
```

**예시**
```json
{
  "success": true,
  "data": {
    "result": "작업 완료",
    "timestamp": "2026-01-01T00:00:00Z"
  }
}
```

**에러 응답**

| 상태 코드 | 에러 코드 | 메시지 | 설명 |
|----------|----------|--------|------|
| 400 | `invalid-input` | 입력값이 올바르지 않다 | 필수 필드 누락 |
| 401 | `unauthorized` | 인증이 필요하다 | 로그인 필요 |
| 403 | `forbidden` | 권한이 없다 | 접근 권한 없음 |
| 404 | `not-found` | 리소스를 찾을 수 없다 | 존재하지 않는 리소스 |
| 500 | `internal-error` | 서버 오류가 발생했다 | 서버 내부 오류 |

**에러 예시**
```json
{
  "success": false,
  "error": {
    "code": "invalid-input",
    "message": "field1은 필수 입력값이다"
  }
}
```

**검증 규칙**
- `field1`: 필수, 1-100자
- `field2`: 필수, 0 이상 정수
- `field3`: 선택, boolean
