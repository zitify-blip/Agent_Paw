Agent Paw UI에서 사용하는 아이콘의 사이즈·스트로크·명명·최적화 기준을 정의한다. Designer는 본 문서 기준에 맞지 않는 외부 아이콘의 직접 사용을 금지한다.

---

## 1. 아이콘 사이즈

모든 아이콘은 픽셀 그리드 정렬(Pixel-snap)을 기준으로 제작한다.

| 토큰 | 크기 | 용도 |
|---|---|---|
| icon/sm | 16px | Inline Text, Badge, Caption |
| icon/md | 20px | Button, Form Field, List Item |
| icon/lg | 24px | Toolbar, Navigation, Card Header |
| icon/xl | 32px | Empty State, Feature Highlight |
| icon/2xl | 48px | Onboarding Illustration |

단, Marketing·Hero 영역의 대형 아이콘은 64px·96px을 별도 토큰 없이 허용한다.

## 2. 스트로크 규칙

아이콘은 Stroke 기반(Outline Style)을 기본 스타일로 사용한다.

| 사이즈 | Stroke Width | Corner Radius |
|---|---|---|
| 16px | 1.5px | 2px |
| 20px | 1.5px | 2px |
| 24px | 2px | 2px |
| 32px | 2px | 3px |
| 48px 이상 | 2.5px | 4px |

. Stroke Cap : `round` 고정
. Stroke Join : `round` 고정
. Fill Style 아이콘 : 활성 상태 강조(Selected Tab 등) 한정 허용

## 3. ViewBox 규격

. 모든 SVG의 viewBox는 `0 0 24 24` 정사각형을 기본으로 한다.
. 내부 드로잉은 20x20 영역 중앙 정렬을 원칙으로 하며, 상하좌우 2px 여백을 확보한다.
. 16px 아이콘은 `0 0 16 16` 별도 viewBox를 사용한다. 24px에서 스케일 다운 사용 불가.

## 4. 명명 규칙

| 구분 | 형식 | 예시 |
|---|---|---|
| Figma 컴포넌트 | `Icon/[Category]/[Name]` | `Icon/Action/Download` |
| SVG 파일명 | `ic-[name]-[variant].svg` | `ic-download-outline.svg` |
| 코드 import | `IconDownload` (PascalCase) | `<IconDownload size={20} />` |

. Name은 kebab-case 영문 소문자를 사용한다. 한글·띄어쓰기·특수문자 사용 불가.
. Variant는 `outline`(기본) / `filled` / `duotone` 3종으로 제한한다.
. 동의어 아이콘(예: Remove·Delete) 중복 제작을 금지하며 하나의 표준명으로 통합한다.

## 5. SVG 최적화

개발 전달 전 Designer는 SVGO 또는 Figma Export Setting을 통해 아래 항목을 준수한다.

| 항목 | 기준 |
|---|---|
| 불필요한 속성 | `id`, `data-*`, `inkscape:*` 제거 |
| Stroke 변환 | Stroke 유지(`vector-effect: non-scaling-stroke` 적용 안함) |
| Fill 색상 | `currentColor` 사용 — CSS `color` 속성으로 제어 |
| Precision | 소수점 2자리 이하 |
| Group·Mask | 단일 `<path>`로 Flatten 권장 |

```svg
<!-- ✅ 권장 -->
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none"
     stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round">
  <path d="M12 3v12m0 0l-4-4m4 4l4-4M4 19h16"/>
</svg>
```

단, 브랜드 로고·다중 색상 일러스트는 `currentColor` 적용 예외 대상으로 분류한다.

## 6. 접근성

. 장식 아이콘 : `aria-hidden="true"` 처리한다.
. 기능성 아이콘(단독 Button) : `aria-label="[동작명]"`을 명시한다. (예: `aria-label="닫기"`)
. Touch Target : 아이콘 Button의 실제 Hit Area는 최소 44x44px을 확보한다. 세부 규칙은 `반응형규칙.md §4` 를 따른다.
