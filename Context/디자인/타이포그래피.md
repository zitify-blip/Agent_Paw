Agent Paw UI의 폰트 패밀리·타입 스케일·행간·자간을 정의한다. Designer는 본 문서에 정의된 스케일 외 임의 폰트 크기 지정을 금지한다.

---

## 1. 폰트 패밀리

System은 운영체제·언어에 따라 자동 매핑한다.

| 구분 | 폰트 | 용도 |
|---|---|---|
| 한글 기본 | Pretendard Variable | 웹·앱 기본 |
| 한글 대체 | 맑은 고딕 (Malgun Gothic) | Windows Fallback, DOCX 출력 |
| 영문·숫자 | Inter | 본문·UI 요소 |
| 모노스페이스 | JetBrains Mono | 코드·숫자 정렬 |

CSS `font-family` 선언 : `"Pretendard Variable", Pretendard, Inter, "맑은 고딕", sans-serif`

## 2. 타입 스케일

본문 기준 폰트 크기는 14px이다. 모든 크기는 8pt grid를 준수한다.

| 토큰 | 크기 | Line-height | Weight | Letter-spacing | 용도 |
|---|---|---|---|---|---|
| display | 40px | 1.2 (48px) | 700 | -0.02em | 랜딩 Hero Headline |
| h1 | 32px | 1.25 (40px) | 700 | -0.015em | Page Title |
| h2 | 24px | 1.33 (32px) | 700 | -0.01em | Section Title |
| h3 | 20px | 1.4 (28px) | 600 | -0.005em | Sub-section Title |
| h4 | 18px | 1.44 (26px) | 600 | 0 | Card Title |
| h5 | 16px | 1.5 (24px) | 600 | 0 | List Header |
| h6 | 14px | 1.57 (22px) | 600 | 0 | Form Label |
| body/lg | 16px | 1.5 (24px) | 400 | 0 | 본문 강조 |
| body/md | 14px | 1.5 (21px) | 400 | 0 | 기본 본문 |
| body/sm | 13px | 1.54 (20px) | 400 | 0 | Secondary Text |
| caption | 12px | 1.5 (18px) | 400 | 0.01em | 보조 설명·Hint |
| overline | 11px | 1.45 (16px) | 600 | 0.08em | Label·Tag |

단, 한글 전용 Heading은 Letter-spacing을 0으로 고정한다. 음수 자간은 영문 혼용 문구에만 적용한다.

## 3. Font Weight 체계

| 토큰 | 수치 | 용도 |
|---|---|---|
| regular | 400 | 본문 기본 |
| medium | 500 | 강조 본문, Button Text |
| semibold | 600 | Heading, Form Label |
| bold | 700 | Display, Page Title |

※ Weight 300 이하 및 800 이상은 사용 불가하다. 가독성 저하 및 랜더링 일관성 문제가 발생한다.

## 4. 줄바꿈·정렬 원칙

. 본문 정렬 : 왼쪽 정렬(Left-align)을 기본으로 한다. 양쪽 정렬(Justify)은 사용 불가하다.
. 한 줄 최대 길이 : Body 기준 45~75자를 권장한다. (약 `max-width: 72ch`)
. 줄바꿈 : Designer는 `\n` 강제 줄바꿈 대신 컨테이너 폭으로 자연 줄바꿈되도록 설계한다.

## 5. 반응형 타입 스케일

Desktop 기준 스케일을 기본값으로 사용하며, 브레이크포인트별 스케일 다운을 아래와 같이 적용한다.

| 토큰 | Desktop | Tablet | Mobile |
|---|---|---|---|
| display | 40px | 36px | 28px |
| h1 | 32px | 28px | 24px |
| h2 | 24px | 22px | 20px |
| h3 | 20px | 18px | 18px |
| body/md | 14px | 14px | 14px |

단, body/md·caption·overline은 브레이크포인트에 무관하게 고정한다.

## 6. 토큰 명명 규칙

. CSS Variable : `--font-size-[token]`, `--line-height-[token]`
. Figma Style : `Type/[Token]` (예: `Type/h1`, `Type/body-md`)
. 개발 대응 Tailwind preset : `text-display`, `text-h1`, `text-body-md`
