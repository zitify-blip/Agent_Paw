Agent Paw UI의 간격(Spacing) 토큰을 8pt grid 기반으로 정의한다. Designer는 본 문서의 토큰 외 임의 픽셀 값을 패딩·마진에 지정하는 것을 금지한다.

---

## 1. 기본 원칙

. 기준 단위 : 8px을 1 grid unit으로 사용한다. (1unit = 8px)
. 예외 단위 : 4px은 Inline Micro-spacing 전용(아이콘-텍스트 간 등)으로 제한 사용한다.
. 금지 값 : 3px·5px·7px 등 grid에 정렬되지 않는 홀수 픽셀 값을 사용 불가하다.
. 반응형 : 브레이크포인트에 따라 동일 토큰의 실제 px 값이 달라지지 않는다. 스케일 필요 시 별도 토큰을 사용한다.

## 2. Spacing 토큰

| 토큰 | 값 (px) | Grid | 주요 용도 |
|---|---|---|---|
| space/0 | 0 | 0 | Reset |
| space/3xs | 2 | - | Hairline Divider |
| space/2xs | 4 | 0.5u | 아이콘-텍스트 간, Tag 내부 |
| space/xs | 8 | 1u | 폼 필드 내부 상하 Padding |
| space/sm | 12 | 1.5u | Button 내부 수직, List Item 간 |
| space/md | 16 | 2u | Card Padding, 기본 Gap |
| space/lg | 24 | 3u | Section 내부 Block 간 |
| space/xl | 32 | 4u | Block 간격(Mobile Section) |
| space/2xl | 48 | 6u | Block 간격(Desktop) |
| space/3xl | 64 | 8u | Section 간(Tablet) |
| space/4xl | 80 | 10u | Section 간(Desktop) |
| space/5xl | 120 | 15u | Landing Hero 외곽 여백 |

## 3. 패딩 원칙

| 컴포넌트 | 수직 Padding | 수평 Padding | 비고 |
|---|---|---|---|
| Button/sm | 8px (space/xs) | 12px (space/sm) | 32px 높이 |
| Button/md | 12px (space/sm) | 16px (space/md) | 40px 높이 |
| Button/lg | 16px (space/md) | 24px (space/lg) | 48px 높이 |
| Input Field | 12px (space/sm) | 16px (space/md) | 44px 높이 |
| Card | 24px (space/lg) | 24px (space/lg) | 균등 |
| Dialog | 24px (space/lg) | 24px (space/lg) | Header·Body·Footer 공통 |
| Table Cell | 12px (space/sm) | 16px (space/md) | 촘촘한 Data Grid는 8px |

## 4. 마진 원칙

. 외부 간격은 마진 대신 부모 Flex·Grid의 `gap` 속성으로 제어하는 것을 우선한다.
. Stack(수직 나열) 컴포넌트는 자식 요소에 마진을 직접 지정하지 않는다.
. 마진을 사용해야 하는 경우 `margin-top`·`margin-bottom` 한 방향만 지정하여 Collapse 예측 가능성을 유지한다.

## 5. Component 간 Gap 권장값

| 관계 | Gap | 토큰 |
|---|---|---|
| 아이콘 ↔ 텍스트 | 4px | space/2xs |
| Form Label ↔ Input | 8px | space/xs |
| Input ↔ Helper Text | 4px | space/2xs |
| Field ↔ Field | 16~24px | space/md ~ space/lg |
| Card ↔ Card | 16px (Mobile) / 24px (Desktop) | space/md / space/lg |
| Section ↔ Section | 48~80px | space/2xl ~ space/4xl |

## 6. 토큰 명명 규칙

. CSS Variable : `--space-[token]` (예: `--space-md` = 16px)
. Figma Variable : `Spacing/[Token]` 컬렉션 (예: `Spacing/md`)
. Tailwind : `p-md`, `gap-lg` 등 커스텀 preset 매핑

단, Figma Auto-layout의 `Spacing between items` 값은 반드시 토큰 Variable를 바인딩하며 하드코딩 픽셀 입력을 금지한다.
