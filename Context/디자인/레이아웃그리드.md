Agent Paw의 레이아웃 그리드·브레이크포인트·컨테이너 규격을 정의한다. Designer는 본 문서 기준 외 임의 컬럼 수·거터 값 지정을 금지한다.

---

## 1. 기본 그리드 체계

. 컬럼 수 : Desktop·Tablet 12 컬럼 / Mobile 4 컬럼
. 거터(Gutter) : 컬럼 간 간격 — 24px(Desktop), 20px(Tablet), 16px(Mobile)
. 마진(Margin) : 컨테이너 외곽 여백 — 64px(Desktop), 32px(Tablet), 16px(Mobile)
. 기본 단위 : 8pt grid를 준수하며 세부 값은 `간격시스템.md §1` 을 따른다.

## 2. 브레이크포인트

System은 `min-width` 미디어 쿼리 기반으로 브레이크포인트를 전환한다.

| 토큰 | Min Width | 컬럼 수 | 거터 | 마진 | 기준 디바이스 |
|---|---|---|---|---|---|
| mobile | 0px | 4 | 16px | 16px | ~ 599px 스마트폰 |
| tablet | 600px | 12 | 20px | 32px | 태블릿·세로 화면 |
| desktop | 1024px | 12 | 24px | 64px | 일반 데스크톱 |
| wide | 1440px | 12 | 24px | auto | 와이드 모니터 |

단, 1920px 이상 초광폭 화면은 컨테이너 max-width로 중앙 정렬하며 그리드 확장을 금지한다.

## 3. 컨테이너 max-width

| 컨테이너 | max-width | 용도 |
|---|---|---|
| container/sm | 640px | Form·Dialog·로그인 등 좁은 영역 |
| container/md | 960px | 일반 Article·Settings Page |
| container/lg | 1200px | Dashboard·List View |
| container/xl | 1440px | 전체 폭 Data Grid·Admin |
| container/full | 100% | Landing Hero·Full-bleed Banner |

. 컨테이너는 부모 기준 `margin-inline: auto`로 가운데 정렬한다.
. Content Width 최대값은 1440px을 기본으로 하며 초과 시 좌우 공백(Letterbox)을 유지한다.

## 4. 컬럼 폭 산정 공식

Desktop 1440px 기준 컬럼 1개의 폭 산정은 아래와 같다.

```
Column Width = (Container Width - (Gutter × (Columns - 1)) - (Margin × 2)) / Columns
             = (1440 - (24 × 11) - (64 × 2)) / 12
             = (1440 - 264 - 128) / 12
             = 87.33px
```

※ Designer는 실제 Figma Auto-layout에서 정수 픽셀을 강제하지 않고 Hug·Fill 속성으로 자동 배분한다.

## 5. Row·Section 간격

| 영역 | 간격 | 기준 |
|---|---|---|
| Section 간 | 80px (Desktop) / 48px (Mobile) | 대주제 구획 |
| Block 간 | 48px (Desktop) / 32px (Mobile) | 소주제 구획 |
| Component 간 | 24px (Desktop) / 16px (Mobile) | Card·List 간 |
| Inline 간 | 8~16px | 텍스트·아이콘 인접 |

## 6. Figma Grid 설정

Designer는 Figma Frame의 Layout Grid를 아래 Preset으로 고정한다.

| Frame | Preset |
|---|---|
| Desktop (1440) | Columns 12 · Type Stretch · Margin 64 · Gutter 24 |
| Tablet (834) | Columns 12 · Type Stretch · Margin 32 · Gutter 20 |
| Mobile (375) | Columns 4 · Type Stretch · Margin 16 · Gutter 16 |

단, 하나의 Frame에 다중 Grid를 중첩하지 않는다. Sub-grid가 필요한 경우 Auto-layout 기반 Nested Frame으로 해결한다.
